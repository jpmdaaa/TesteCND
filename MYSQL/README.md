# Banco de Dados MySQL - Análise e Recomendações de Filmes

Este repositório contém cinco scripts para interação com um banco de dados MySQL. Eles estão divididos entre **escrita** (inserção de dados) e **leitura** (consulta e análise de dados).

## **Estrutura dos Scripts**

| Script                               | Função                                  |
| ------------------------------------ | ----------------------------------------- |
| **`Connect.sql`**            | Criação do banco de dados e tabelas     |
| **`analise_WRITE.py`**       | Insere dados de análise no banco         |
| **`analise_READ.py`**        | Lê e analisa dados do banco              |
| **`recomendacoes_WRITE.py`** | Insere recomendações de filmes no banco |
| **`recomendacoes_READ.py`**  | Lê e exibe recomendações               |
